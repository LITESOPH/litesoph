============================
 LITESOPH
============================


Requirements
============

  * Python 3.8 or higher
  * Tkinter
  * click
  * Numpy
  * Matplotlib
  * Paramiko

Installation
============

.. code-block:: console

  $ git clone https://github.com/13sachin/litesoph.git
  $ pip install litesoph


Usage
=====

To start gui application, run:

.. code-block:: console

  $ litesoph gui
