============================
 LITESOPH
============================


Requirements
============

  * Python 3.7 or higher
  * Tkinter
  * click
  * Numpy
  * Matplotlib
  * Paramiko

Installation
============

  $git clone https://github.com/13sachin/litesoph.git
  $pip install litesoph


Usage
=====

To start gui application, run::

  $litesoph gui
